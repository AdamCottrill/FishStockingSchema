* Fish Stocking Schema

This repository contains a minimal working example that is intended to
demonstrate how Pydantic could be used to validate the data contained
in a GLFC Fish Stocking data submission.  It contains a basic
validator for fish stocking data as well as the code to fetch current
lookup values and read in the data from a submitted spreadsheet.

Currently, the script fetches the data from the specified excel spreadsheet
The validated data and any validation errors are accumulated in an array that is
reported at the end of the script.

This is a minimal, proof of concept.  Additional improvements include
updates to the validator, tests, more meaningful error reports, and a better
user interface (either as a desktop application or a web based
service).

** Installation

Create and activate python virtual environment:

#+begin_src bash

  mkvirtualenv fish_stocking_schema
  workon fish_stocking_schema

#+end_src

Install the required packages:

#+begin_src bash

  pip install -r requirements.txt

#+end_src


Update the SRC_DB variable in the validate_stocking_data.py script to point to
a populated Fish Stocking Submission Template on your computer (currently
only xlsx files are supported).  Save the file, and then run the python
script in a terminal:


#+begin_src bash

  python validate_stocking_data.py

#+end_src
